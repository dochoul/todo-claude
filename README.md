# Todo App

React + TypeScript로 만든 할일 관리 앱입니다.

## 기능

### 할일 관리
- **추가**: 내용 입력 후 Enter 또는 추가 버튼 클릭
- **완료 체크**: 체크박스 클릭으로 완료/미완료 전환
- **삭제**: × 버튼으로 항목 삭제

### 카테고리
개인 / 업무 / 쇼핑 / 기타로 분류하고 색상 배지로 구분합니다.

### 중요도
높음 / 보통 / 낮음 3단계로 설정합니다.
- 카드 왼쪽 테두리 색상으로 한눈에 구분
- 색상 배지(🔴 🟡 🟢)로 추가 표시

### 마감일 (Due Date)
- 할일 추가 시 마감일 선택 가능 (선택사항)
- D-Day 배지로 남은 일수 표시
  - `D-3` 이상: 회색
  - `D-1` ~ `D-2`: 주황색 (임박)
  - `D-Day`: 빨간색
  - 기한 초과: 진한 빨강 배경 (`D+N`)
- **오늘 마감** 필터로 오늘 처리할 항목만 빠르게 확인

### 필터
| 버튼 | 설명 |
|------|------|
| 전체 | 모든 할일 표시 |
| 오늘 마감 | 오늘이 마감인 항목만 표시 |
| 개인 / 업무 / 쇼핑 / 기타 | 카테고리별 필터 |

### 데이터 저장
브라우저 localStorage에 자동 저장됩니다. 새로고침해도 데이터가 유지됩니다.

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

| 항목 | 내용 |
|------|------|
| 프레임워크 | React 18 |
| 언어 | TypeScript 5 |
| 빌드 도구 | Vite 5 |
| 스타일 | CSS Modules (순수 CSS) |
| 상태 관리 | React useState + useMemo (외부 라이브러리 없음) |
| 데이터 저장 | localStorage |

## 프로젝트 구조

```
src/
├── types/
│   └── todo.ts              # 타입 정의 + 마감일 헬퍼 함수
├── hooks/
│   └── useTodos.ts          # 상태 관리 커스텀 훅
└── components/
    ├── AddTodo/             # 할일 입력 폼
    ├── TodoList/            # 할일 목록
    ├── TodoItem/            # 할일 카드 (D-Day 배지 포함)
    └── CategoryFilter/      # 필터 버튼 목록
```
